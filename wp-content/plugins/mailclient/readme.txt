== Description ==
Free oMailClient (FMC) is a:
Small plugin, with a simple way to display Webmail on the backend of your website.
You can recieve, delete and read e-mails in the backend of your site.
It is still under development and will become extra features like, recieving mail
from pop3 and imap mailserver protocols.
Shortcode's, widgets possibilities will added to.

So check on regular basis for new version!

== Installation ==
Just upload zip file with your plugin manager and activate plugin.
Initiation is done in backend, there you have to supply:
    mailserver: mail.yourserver.com
    user      : your username
    pass      : your password, for that mail account
